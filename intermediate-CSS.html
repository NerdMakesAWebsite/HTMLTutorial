CSS Intermediate Tutorial

Learn about layout techniques, pseudo-classes, and responsive design.

1. Flexbox Layout

Flexbox helps create flexible layouts:

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

2. Pseudo-Classes

Pseudo-classes add special effects to elements, such as :hover:

a:hover {
    color: red;
    text-decoration: underline;
}

3. Media Queries

Use media queries for responsive design:

@media (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}
