CSS Advanced Tutorial

Dive deeper into animations, grid layouts, and CSS variables.

1. CSS Animations

Create animations using the @keyframes rule:

@keyframes fadeIn {
    from { opacity: 0; }
    to { opacity: 1; }
}

.fade-in {
    animation: fadeIn 2s ease-in-out;
}

2. Grid Layout

Use CSS Grid for complex layouts:

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

3. CSS Variables

Use variables to simplify and maintain styles:

:root {
    --primary-color: #3498db;
}

button {
    background-color: var(--primary-color);
    color: white;
}
