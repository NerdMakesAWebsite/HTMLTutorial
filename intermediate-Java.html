JavaScript Intermediate Tutorial

Learn about objects, arrays, and basic DOM manipulation.

1. Arrays

Store multiple values in an array:

let fruits = ["apple", "banana", "cherry"];
console.log(fruits[0]); // Outputs "apple"

2. Objects

Objects hold key-value pairs:

let person = {
    firstName: "John",
    lastName: "Doe",
    age: 30
};
console.log(person.firstName); // Outputs "John"

3. DOM Manipulation

Change the content of a webpage using the DOM:

document.getElementById("myDiv").textContent = "Hello, JavaScript!";
