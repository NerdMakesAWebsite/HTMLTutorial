JavaScript Basic Tutorial

Learn the basics of JavaScript, including variables, functions, and events.

1. Variables

Variables store data values:

let name = "John";
const age = 30;
var isStudent = true;

2. Functions

Functions perform actions:

function greet() {
    console.log("Hello, world!");
}
greet();

3. Events

Handle user actions with events:

document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});
