HTML Intermediate Tutorial

Expand your HTML knowledge with lists, tables, forms, and attributes.

1. HTML Lists

There are two main types of lists in HTML: ordered (<ol>) and unordered (<ul>):

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First</li>
    <li>Second</li>
</ol>

2. HTML Tables

Tables allow you to display data in a tabular format:

<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>24</td>
    </tr>
</table>

3. Forms and Input Elements

Forms are used to collect user input. Here's how to create a basic form:

<form action="/submit" method="POST">
    <label for="username">Username:</label>
    <input type="text" id="username" name="username">
    <br>
    <label for="password">Password:</label>
    <input type="password" id="password" name="password">
    <br>
    <input type="submit" value="Submit">
</form>

4. HTML Attributes

Attributes provide additional information about HTML elements. Common attributes include id, class, style, and title:

<div id="main" class="container" title="Main Container">Content</div>
<p style="color: red;">This is a styled paragraph.</p>
