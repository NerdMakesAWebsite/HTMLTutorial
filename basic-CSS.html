CSS Basic Tutorial

Learn the basics of CSS, including how to style text, colors, and backgrounds.

1. Styling Text

Use the color, font-size, and font-family properties to style text:

p {
    color: blue;
    font-size: 18px;
    font-family: Arial, sans-serif;
}

2. Colors and Backgrounds

Change text and background colors using the background-color and color properties:

body {
    background-color: #f0f0f0;
}

h1 {
    color: #333;
    background-color: yellow;
}

3. Borders and Padding

Adjust borders, padding, and margins to add space around elements:

div {
    border: 2px solid black;
    padding: 10px;
    margin: 20px;
}
