HTML Advanced Tutorial

In this advanced section, you'll learn about semantic HTML, forms, multimedia, and working with APIs.

1. Semantic HTML

Semantic HTML elements clearly describe their meaning. These elements help with accessibility and SEO:

<header>Website Header</header>
<nav>Navigation Menu</nav>
<article>Main Content</article>
<footer>Website Footer</footer>

2. Forms and Input Validation

HTML forms allow users to submit data. You can also add validation to ensure data is correct before submission:

<form action="/submit" method="POST">
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    <input type="submit" value="Submit">
</form>

3. Multimedia: Audio and Video

HTML supports multimedia elements, such as audio and video, to make your website more interactive:

<audio controls>
    <source src="audio.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

<video controls width="320" height="240">
    <source src="video.mp4" type="video/mp4">
    Your browser does not support the video tag.
</video>

4. Working with APIs

You can fetch data from external sources using HTML and JavaScript. This example shows how to fetch and display data from an API:

<script>
    fetch('https://api.example.com/data')
        .then(response => response.json())
        .then(data => {
            console.log(data);
            document.getElementById('api-output').textContent = JSON.stringify(data);
        })
        .catch(error => console.error('Error:', error));
</script>

<div id="api-output">API data will appear here</div>
