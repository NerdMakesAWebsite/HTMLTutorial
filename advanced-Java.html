JavaScript Advanced Tutorial

Learn advanced concepts like asynchronous programming, closures, and ES6 features.

1. Promises and Async/Await

Handle asynchronous operations:

async function fetchData() {
    let response = await fetch("https://api.example.com/data");
    let data = await response.json();
    console.log(data);
}
fetchData();

2. Closures

Understand closures and their use cases:

function makeCounter() {
    let count = 0;
    return function() {
        return ++count;
    };
}

let counter = makeCounter();
console.log(counter()); // Outputs 1
console.log(counter()); // Outputs 2

3. ES6 Features

Explore new JavaScript features such as arrow functions, template literals, and destructuring:

// Arrow function
const greet = () => "Hello, ES6!";

// Template literals
let name = "John";
console.log(`Hello, ${name}!`);

// Destructuring
let [a, b] = [1, 2];
console.log(a); // Outputs 1
