HTML Basic Tutorial

Learn the fundamentals of HTML, including basic structure, text formatting, and links.

1. HTML Structure

Every HTML document starts with a doctype declaration and has a basic structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Page Title</title>
</head>
<body>
    <h1>Welcome to HTML!</h1>
    <p>This is a paragraph.</p>
</body>
</html>

2. Text Formatting

HTML provides tags for different types of text formatting:

<b>Bold Text</b>
<i>Italic Text</i>
<u>Underlined Text</u>
<br> 
<hr>

3. Links and Navigation

Links are created using the <a> tag, allowing users to navigate between pages:

<a href="https://www.example.com">Visit Example</a>
<a href="page.html">Go to another page</a>

4. Images

You can add images using the <img> tag. Make sure to provide the src attribute for the image location:

<img src="image.jpg" alt="A description of the image">
